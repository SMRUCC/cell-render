modelling_cellgraph {CellRender} R Documentation

Run the cell graph structure modelling workflow

Description

Usage

modelling_cellgraph(src,
    outputdir = NULL,
    up.len = 150,
    biocyc = './biocyc',
    regprecise = './RegPrecise.Xml');

Arguments

src

the ncbi genbank assembly source file its file path

outputdir

the result outputdir and temp workspace location, default value is NULL means use the parent dir of the input src

biocyc

the directory path to the biocyc reference pathway

regprecise

the data repository xml file path to the regprecise motif database.

Details

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

this function has no return value, and then generated virtual cell model file will be saved at the outputdir with a fixed file name: model.vcell. and the output dir also contains a html/pdf report about the virtual cell modelling result.
