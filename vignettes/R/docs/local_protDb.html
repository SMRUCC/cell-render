local_protDb {CellRender} R Documentation

Create local fasta database for protein alignment

Description

Usage

local_protDb(cad.registry, dbfile);

Arguments

Details

this function will pull all protein sequence from the biocadregistry, and then build local fasta sequence database with header formats: > cad_id ec_number|function description ` a placeholder symbol - will be placed after the cadid if the ec_number of the protein is missing. for multiple ecnumber, a comma seperator will be used for join the ecnumber list.

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

This function has no value returns.
