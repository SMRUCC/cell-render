list_batch_models {CellRender} R Documentation

List All Batch Model Directories

Description

Retrieves the list of model subdirectories from the release directory configured in the current workflow session. Each subdirectory represents a separate virtual cell model that was created during batch processing.

Usage

list_batch_models(
    tqdm.list = FALSE);

Arguments

Details

This function reads the \code{release} configuration parameter to determine the root directory, then lists all immediate subdirectories. Each subdirectory is expected to contain a \code{builder.gcproj} file produced by \code{\link{makegenbankproj_file}}.

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

A character vector of directory paths, one per model. Returns an empty vector if no model directories exist.

Examples

#\dontrun{

# List all models in the release directory
models <- list_batch_models()
print(models)
# [1] "/path/to/release/GCF_000005845" "/path/to/release/GCF_000006945"

#}
