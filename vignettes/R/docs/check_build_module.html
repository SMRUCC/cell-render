check_build_module {CellRender} R Documentation

Check Whether a Build Module is Enabled

Description

Checks if a specific virtual cell build module is enabled in the current workflow configuration. This is used to conditionally execute workflow steps based on user-specified build targets.

Usage

check_build_module(flag);

Arguments

flag

Character. The name of the build module to check (e.g., \code{"TRNnetwork"}, \code{"metabolicnetwork"}). The comparison is case-insensitive.

Details

The function reads the \code{builds} configuration parameter (a character vector of enabled module names) and performs a case-insensitive match against the provided \code{flag}.

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

Logical. \code{TRUE} if the specified module is present in the \code{builds} configuration, \code{FALSE} otherwise.

Examples

#\dontrun{

# Check if the TRN network module should be built
if (check_build_module("TRN_network")) {
message("TRN network module is enabled")
}

#}
