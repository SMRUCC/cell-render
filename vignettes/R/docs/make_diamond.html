make_diamond

Description

Generates DIAMOND binary databases (`.dmnd`) from source FASTA files. This utility function constructs three specific databases required for the annotation workflow: Enzyme Commission (EC) numbers, Subcellular locations, and Transcription Factors.

Usage

make_diamond(local.db,
    diamond = 'Sys.which'('diamond'));

Arguments

local.db

A character string specifying the directory path where the source FASTA files are located.

diamond

A character string specifying the path to the DIAMOND executable. Defaults to \code{Sys.which("diamond")} if not provided.

Details

This function expects the following source files to exist in \code{local_db}: \itemize{ \item \code{ec_numbers.fasta} \item \code{subcellular.fasta} \item \code{TF.fasta} } It utilizes \code{system2} to invoke the \code{diamond makedb} command.

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

Invisible \code{NULL}. This function is called for its side effect of creating database files (e.g., \code{ec_number.dmnd}) in the current working directory.
