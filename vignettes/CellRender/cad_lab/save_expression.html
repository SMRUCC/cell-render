save_expression {cad_lab} R Documentation

save the molecule expression data into the database

Description


The molecule expression data is a Matrix object, which contains the molecule expression
data for each gene in the virtual cell. The cad_lab object is used to access the database
and save the data.

The function will create a new experiment project if it does not exist, and then save the molecule
expression data into the database.

Usage

save_expression(cad.lab, exp.id, dynaimics);

Arguments

cad.lab

[as cad_lab]

exp.id

the experiment project id. [as string]

dynaimics

[as Matrix]
